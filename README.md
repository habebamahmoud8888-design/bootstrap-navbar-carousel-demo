# Bootstrap Navbar & Carousel Demo

This project was built as part of my journey learning front-end development.
It focuses on getting comfortable with Bootstrap 5 by implementing the kind of
UI components that appear in most real-world websites.

## What I Built

The page includes a responsive navbar that collapses on smaller screens,
dropdown menus both inside and outside the navbar, an inline search bar,
and an image carousel with custom indicators and navigation controls.

The goal was to understand how these components behave and interact
without writing any custom JavaScript — relying entirely on Bootstrap's
built-in functionality.

## Built With

- [Bootstrap 5.3.8](https://getbootstrap.com/)
- HTML5 and CSS3
- [Lorem Picsum](https://picsum.photos/) for placeholder images
